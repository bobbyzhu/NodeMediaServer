# About Node Media Server
Node Media Server是一个由JavaScript编写的流媒体服务端，目前支持RTMP协议。
使用Fibjs驱动 https://github.com/xicilion/fibjs

 - Supports only RTMP.
 - Supports only H.264 video and AAC audio.
 - Supports publish/play multiple streams.

# Usage
    fibjs app.js
